# Sim AI Free

Free AI Agent Platform - clone of sim.ai

## Features
- AI Assistant (Gemini 2.5 Flash) - free, no API key needed from user
- Visual workflow editor
- Tables, Files, Knowledge Base
- 150+ integrations
- Completely free forever
- No registration, no subscriptions

## How to run
Open index.html in browser or deploy to GitHub Pages.

## Deploy to GitHub Pages
1. Go to repo Settings > Pages
2. Source: Deploy from branch
3. Branch: main, folder: / (root)
4. Save
